2️⃣ What is the spread operator (...)?
  Ans : The spread operator is an operator that spreads the data of an array or object one by one. It is usually used to expand,  copy, or pass data as arguments.

3️⃣ What is the difference between map(), filter(), and forEach()?
  Ans : The difference between map(), filter(), and forEach() is :
      map() : Takes one element from the original array, performs a specific operation, and returns the data in a new array.Does not modify the original array.
      filter() : Takes an element from the original array and returns those elements in a new array that meets the specified condition.Does not modify the original array.
      forEach() : Takes an element from an array and performs the specified operation. It does not return anything. It modifies the original array.

4️⃣ What is an arrow function?
  Ans : Arrow functions are a feature of ES6 that allows you to write functions very quickly and concisely. The =>' symbol is used to write arrow functions.

5️⃣ What are template literals?
